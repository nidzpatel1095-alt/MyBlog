# MyBlog – Smart Content Sharing Platform 🚀

### 📌 Project Overview
MyBlog is a full-stack blogging web application developed for the Software Engineering (3161605) CIPAT task. The platform provides a professional environment for users to share stories across categories like Technology, Food, and Travel.

---

### 🛠️ Tech Stack
* Frontend: HTML5, CSS3, JavaScript (AJAX)
* Backend: PHP (Server-side logic)
* Database: MySQL
* Tools: XAMPP Server, Git, GitHub

---

### ✨ Key Features
* User Authentication: Secure registration and login system.
* Content Management: Full CRUD (Create, Read, Update, Delete) for blog posts.
* Media Support: Image upload functionality.
* Real-time Engagement: AJAX-based "Like" and "Comment" system.
* Admin Dashboard: Centralized panel for moderation and user management.

---

### 📈 SDLC & Industry Practices Applied
As per the CIPAT guidelines, this project follows industry-standard practices:
1. Agile Methodology: Iterative development through feature-based sprints.
2. UML Modeling: Systematic design using Use Case, Class, and Sequence diagrams.
3. Configuration Management: Version control using Git with branching (develop and main) and merging strategies.
4. Quality Assurance: Systematic manual testing with defined test cases.

---

### 🚀 How to Run the Project
1. Clone the repository: `git clone https://github.com/nidzpatel1095-alt/MyBlog.git`
2. Move the project folder to `C:/xampp/htdocs/`.
3. Import the database file into phpMyAdmin.
4. Start Apache and MySQL from XAMPP.
5. Open browser and go to `localhost/MyBlog`.
