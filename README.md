# MyBlog – Smart Content Sharing Platform 🚀

### 📌 Project Overview
[cite_start]**MyBlog** is a full-stack blogging web application developed for the **Software Engineering (3161605) CIPAT** task[cite: 2, 3]. The platform provides a professional environment for users to share stories across categories like Technology, Food, and Travel.

---

### 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript (AJAX)
* **Backend:** PHP (Server-side logic)
* **Database:** MySQL
* [cite_start]**Tools:** XAMPP Server, Git, GitHub [cite: 8, 20]

---

### ✨ Key Features
* [cite_start]**User Authentication:** Secure registration and login system[cite: 11, 47].
* [cite_start]**Content Management:** Full CRUD (Create, Read, Update, Delete) for blog posts[cite: 11, 49].
* **Media Support:** Image upload functionality.
* **Real-time Engagement:** AJAX-based "Like" and "Comment" system.
* **Admin Dashboard:** Centralized panel for moderation and user management.

### 🚀 How to Run the Project
1. Clone the repository: `git clone https://github.com/nidzpatel1095-alt/MyBlog.git`
2. Move the project folder to `C:/xampp/htdocs/`.
3. Import the database file into **phpMyAdmin**.
4. Start Apache and MySQL from XAMPP.
5. Open browser and go to `localhost/MyBlog`.
